I am Michael Uzan, a software developer with interest in crypto and digital trading. In this project, I developed the Coin's Application to display cryptocurrency data, including live prices and graphical reports.

Project Description

The application uses an external API to fetch coin data and presents it interactively. You can search coins by symbol, track selected coins, and view live reports with up-to-date charts.
